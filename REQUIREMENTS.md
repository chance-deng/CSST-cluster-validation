# Requirements

The workflow was run with Python 3.9 on macOS. Core packages used across the scripts include:

```text
numpy
pandas
matplotlib
scipy
astropy
scikit-image
scikit-learn
torch
```

Optional but useful:

```text
h5py
tqdm
```

Plot style follows the project convention: Times New Roman, Nature-like clean styling, `dpi=300` for saved figures.
